# Flora Documentation

This folder is organized by topic so it is easier to scan and maintain.

## Reading Guide

- Start with [architecture/FLORA_HIGH_LEVEL_ARCHITECTURE.puml](./architecture/FLORA_HIGH_LEVEL_ARCHITECTURE.puml) for the system overview.
- Use [deployment/DEPLOYMENT.md](./deployment/DEPLOYMENT.md) for day-to-day deploy steps.
- Use [testing/TESTING_GUIDE.md](./testing/TESTING_GUIDE.md) for demo checks and subscription test flows.
- Use [integrations/](./integrations) when working on Auth0, Stripe, Gemini, subscriptions, or email-related behavior.

## Folder Structure

### [architecture/](./architecture)

- Includes the main PlantUML file and supporting flow references

### [deployment/](./deployment)

### [infrastructure/](./infrastructure)

### [integrations/](./integrations)

### [testing/](./testing)
- Manual and automated test guides
- Demo-day scenarios and validation checklists
